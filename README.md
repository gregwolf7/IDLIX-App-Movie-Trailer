# IDLIX-App-Movie-Trailer
IDLIX adalah aplikasi android ringan yang dirancang khusus untuk para penggemar film agar dapat menjelajahi informasi film terbaru dengan cepat, aman, dan tanpa gangguan. Dapatkan akses instan ke trailer film berkualitas tinggi dan sinopsis lengkap untuk membantu Anda memutuskan film apa yang akan ditonton selanjutnya.
